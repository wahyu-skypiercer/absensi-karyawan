
## cara install
1. composer install
2. npm install
3. php artisan key:generate
4. php artisan migrate --seed
5. php artisan migrate:fresh --seed
6. npm install @vitejs/plugin-react@5
7. npm install chart.js react-chartjs-2

## cara run
1. rm -rf node_modules
2. npm install
3. npm run dev -> jika tidak bisa coba yang ke 4
4. php artisan serve
